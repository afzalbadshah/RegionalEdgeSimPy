RegionalEdgeSimPy — How to Write Your Own Scheduler

Author: Dr. Afzal Badshah • Department of Software Engineering, University of Sargodha

A scheduler decides which server (Edge / Regional / Cloud) each task should run on. To plug in your own logic, create a class that inherits BaseScheduler and implement a single method: schedule(self, tasks, servers, current_time).

Minimal Example

# scheduler/my_scheduler.py
from scheduler.base_scheduler import BaseScheduler

class MyScheduler(BaseScheduler):
    def schedule(self, tasks, servers, current_time):
        assignments = []
        for task in tasks:
            for srv in servers:
                if srv.can_allocate(task.cpu_demand, task.data_size_kb, task.memory_demand):
                    task.set_server(srv.name)
                    assignments.append((task, srv))
                    break
        return assignments

Register Your Scheduler

from scheduler.my_scheduler import MyScheduler
from core.simulator import Simulator

def main():
    scheduler = MyScheduler()
    simulator = Simulator(scheduler=scheduler)
    simulator.run()
