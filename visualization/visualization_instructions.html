RegionalEdgeSimPy — How to Visualize Your Results

Author: Dr. Afzal Badshah • Department of Software Engineering, University of Sargodha

The simulator writes a CSV at results/task_metrics_log.csv. You can customize plots by changing which columns to read and how to group them. Below are simple, beginner-friendly templates using pandas and matplotlib.

1) Quick start: line plot over rounds

Plot CPU utilization by paradigm (Edge/Regional/Cloud) across rounds.

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("results/task_metrics_log.csv")  # columns like: Round, CPU (%), Paradigm, Workload, Devices

metric = "CPU (%)"              # <— change to "Memory (%)", "Conges(%)", "Energy", etc.
x_col  = "Round"                # <— change to "Workload" or "Devices" to match your paper
group  = "Paradigm"             # Edge_1, Regional_1, Cloud_1...

plt.figure()
for name, g in df.groupby(group):
    plt.plot(g[x_col], g[metric], label=name, marker='o')

plt.xlabel(x_col)
plt.ylabel(metric)
plt.legend(title=group)
plt.title(f"{metric} vs {x_col}")
plt.tight_layout()
plt.savefig("visualization/line_{}_by_{}.png".format(metric.replace(" ", "_"), x_col))
plt.show()

2) Switching x-axis to workload or devices

Change x_col to "Workload" (KB) or "Devices":

x_col = "Workload"  # or "Devices"
metric = "Avg_Tx(ms)"  # example: transmission delay
# re-run the plotting loop above

3) Filtering tiers (e.g., only Edge)

df_edge = df[df["Paradigm"].str.startswith("Edge")]
# Then group by Edge servers only
for name, g in df_edge.groupby("Paradigm"):
    plt.plot(g["Round"], g["CPU (%)"], label=name)

4) Bar chart: average metric per paradigm

Average your chosen metric across all rounds and compare paradigms.

metric = "Energy"
avg = df.groupby("Paradigm")[metric].mean().reset_index()

plt.figure()
plt.bar(avg["Paradigm"], avg[metric])
plt.ylabel(f"Average {metric}")
plt.xticks(rotation=20)
plt.title(f"Average {metric} by Paradigm")
plt.tight_layout()
plt.savefig("visualization/bar_avg_{}.png".format(metric))
plt.show()

5) Multi-axis example: workload vs delays

Plot both transmission and propagation delays against workload.

x = "Workload"
y1 = "Avg_Tx(ms)"
y2 = "Avg_Prop(ms)"

plt.figure()
plt.plot(df[x], df[y1], marker='o', label=y1)
plt.plot(df[x], df[y2], marker='s', label=y2)
plt.xlabel(x)
plt.ylabel("Delay (ms)")
plt.legend()
plt.title("Delays vs Workload")
plt.tight_layout()
plt.savefig("visualization/delays_vs_workload.png")
plt.show()

6) Common column names

  • Round, Devices, Workload, Paradigm
  • CPU (%), Memory (%), Storage (%), Conges(%)
  • Avg_Tx(ms), Avg_Prop(ms)
  • Tx_Cost, Proc_Cost, Energy

7) Tips

  • Make sure the CSV path is correct: results/task_metrics_log.csv.
  • Create the folder visualization/ before saving figures.
  • Rename axes to match your paper (e.g., "Workload (KB)").
  • Filter by Paradigm if you want separate charts for Edge/Regional/Cloud.
